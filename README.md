# lleon
这是我的[博客](lleon.xyz)的源码，欢迎交流和Fork
### 介绍
- 前端用Vue.js，后端用的php
- 用vue-router进行路由切换，vue-resource异步获取资源，vuex进行状态管理
- marked进行markdown文章解析
- GFM风格的markdown样式
- highlight.js完成语法高亮
- 整体样式风格借鉴自[码志](https://github.com/mzlogin/mzlogin.github.io)，稍作修改成了扁平化简约风
